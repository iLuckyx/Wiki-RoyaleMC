# Seismic Axe

The Seismic Axe, also known as the portable Earthquake, launches blocks in the air in the direction being looked at when used. This will push any entity within the Earthquake up to 4 blocks away. To use the ability, simply right-click with the axe.

![Seismic Axe](<../../../.gitbook/assets/image (106).png>)

{% hint style="info" %}
Each use of the Earthquake ability consumes 4 points of durability. Spam using this ability will deplete the item extremely quickly.
{% endhint %}
